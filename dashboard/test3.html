Chart.js exemples

Chart.js est une bibliothèque Javascript permettant de créer des graphiques web. Côté technologie, la librairie se repose sur du HTML5 Canvas pour le rendu des graphes.

Les 6 types de graphiques de Chart.js sont Doughnut, Pie, Line, Bar, Radar, Polar Area.
